[编辑] 生成 build.xml 和 local.properties

android update project -p ./ 这个工具用来生成本地的local.properties 和 build.xml

[编辑] 添加属性到 local.properties

key.store=zeleisoft_kidsedu.keystore
key.alias=mykey
key.store.password=xxxx
key.alias.password=xxxx

[编辑] ant release

执行该命令就可以生成release的包。


[编辑] 常遇见问题：

[编辑] build.xml:698: null returned: 1

invalid resource directory name: /home/linkycj/workspace/android/util/bin/res/crunch

解决办法：一般执行ant clean release即可，或者删除这个目录。

[编辑] build.xml:1151: Cannot recover key

一般是因为 key.alias.password= 最后可能有回车或者空格影响的。 解决办法： 

#!/bin/sh
keyPass=tiaditsab
sed -i "s|key.store.password=.*|key.store.password=${keyPass}|g" local.properties
sed -i "s|key.alias.password=.*|key.alias.password=${keyPass}|g" local.properties
ant release

[编辑] 库项目R.java问题

一般是因为混淆了代码的缘故。 proguard-project.txt 添加： -ignorewarnings 但是是否会导致调用的时候出问题？

[编辑] 加密版本问题

jdk1.6和jdk1.7使用的加密算法不同，会因此导致运行时出错。

[编辑] 本地库.so文件的加载问题

会自动将.so拷贝到apk


[编辑] 早教应用打包脚本

#!/bin/sh
 
local_pro="local.properties"
 
SYMBOL="..................................................."
 
libs="encrypt mediaplayer zlsitemmodel zlsindexactivity zlsindexmodel zlstaghtml zlstoolbar ebookbag/ebagcommon hanyupinyin libraries"
others="formula hanzi poem/poemlib poem/song poem/tang storycastle/ cartoonpinyin picturedictionary"
stories="story/abstractstory story/adagestory story/allusionstory story/idiomstory story/slangstory"
enlights="enlights/dizigui enlights/familynames enlights/lunyu enlights/qianziwen enlights/sanzijing enlights/zengguangxianwen"
 
build() {
        packages="$@"
        echo build $packages
        for i in $packages; do
                echo build $i $SYMBOL
                if [ -d $i ]; then
                        cd $i
                        echo rm -rf $local_pro $SYMBOL
                        rm -rf $local_pro
 
                        echo android update project -p ./
                        android update project -p ./
 
                        echo rm -rf proguard-project.txt $SYMBOL
                        rm -rf proguard-project.txt
 
                        echo "-ignorewarnings" >> proguard-project.txt
 
                        #               echo "key.store=/home/linkycj/android/apk/zeleisoft_kidsedu.keystore" >> $local_pro
                        #               echo "key.alias=mykey" >> $local_pro
                        #               echo "key.store.password=xxxx" >> $local_pro
                        #               echo "key.alias.password=xxxx" >> $local_pro
 
                        echo rm -rf $i/bin/res/crunch
                        rm -rf $i/bin/res/crunch
 
                        echo ant clean release $SYMBOL
                        ant clean release > log
 
                        fail=`cat log | grep "BUILD FAILED"`
                        if [ XXX$fail = XXX ]; then
                                echo "ant clean release $i BUILDSUCCESSFULL"                                                                                                                            
                       else
                                echo "ant clean release $i BUILD FAILURE"
                                cat log
                                cd -
                                exit 0
                        fi
 
                        cd -
                fi
        done
}
 
 
if [ XXX$1 = XXX ]; then
        echo build all
        build "$libs"
        build "$stories"
        build "$enlights"
        build $others
elif [ $1 = "libs" ]; then
        echo build $1;
        build $libs
elif [ $1 = "others" ]; then
        echo build $1;
        build $others;
elif [ $1 = "stories" ]; then
        echo build $1;
        build $stories;
elif [ $1 = "enlights" ]; then
        echo build $1;
        build $enlights;
fi
 
 
 
echo cp `find -name "-release.apk"` ~/android/apk/
#cp `find -name "-release.apk"` ~/android/apk/
